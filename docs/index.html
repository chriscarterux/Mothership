What It Does

Auto-Planning

Cipher reads your docs and creates stories. No manual PRD writing.

Loop Until Done

Fresh context each iteration. Never loses track. Runs until complete.

Test Generation

Cortex writes tests for edge cases: null inputs, errors, boundaries.

Code Review

Sentinel checks security, patterns, and quality before merge.

Any AI Tool

Works with Claude, AMP, Cursor, Aider. Not locked to one provider.

Linear or JSON

Store stories in Linear or local JSON. Simple config.

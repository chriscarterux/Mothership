# Mothership diagrams (Mermaid -> Excalidraw)

These Mermaid files are sized for Excalidraw's Text-to-Diagram flow. Paste any file into Excalidraw's Mermaid tab to generate a diagram, then export as PNG/SVG for social sharing.

## Quick use (Excalidraw UI)

1) Open Excalidraw (local app or excalidraw.com).
2) Open the command palette, pick "Mermaid to Excalidraw".
3) Paste the Mermaid content from a file in `diagrams/mermaid/`.
4) Export as PNG/SVG.
